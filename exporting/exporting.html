Save As SVG Export

Once selections have been made, to run Save As SVG, press OK. You will then be prompted to select the location to save the SVG file. This includes the directory path and file name. The default directory is the user’s Downloads folder and the default file name will be based on the model name and sketch name. Click Save to complete the export process.
